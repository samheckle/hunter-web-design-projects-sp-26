## Project #3

Think about post-internet and futuristic networks to build a speculative communication network. How might communication change in our existing world, or how does communication happen on an alien planet? This can be based on real communication networks or ones that don't exist. Think about the affordances of the web and how you can create communications through different types of interactions that are not limited to language!

The only technical requirement is to use Nunjucks and `response.render()`, but there are no other limitations from there.
